# <a name="community-contributor-hall-of-fame"></a>殿堂入りコミュニティ共同作成者

PowerShell コミュニティは、活気のあるコラボレーション グループです。 Microsoft では、コミュニティからのヘルプとサポートに心から感謝しております。 貢献する方法については、[共同作成者ガイド][contrib]をお読みください。
